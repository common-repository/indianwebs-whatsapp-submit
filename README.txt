Plugin para facilitar a los usuarios un contacto más directo mediante Whats de una manera muy simple e intuitiva.

== Description ==

Actualmente, ya esta quedando obsoleto el uso de formularios en las paginas, ya que, todo el mundo hace uso de la red social Whats, y por eso, tenemos que aprovechar su uso y facilidad.

Nosotros, venimos a ofrecerte, con una configuración muy simple e intuitiva, un botón de Whats para el envío de mensajes, donde puedes limitar su uso por horas, seleccionar su posición, establecer un mensaje por defecto y una llamada a la acción.

Estas son las funciones que añade el plugin:

*   "Número de teléfono" El icono soló se mostrara si se ha establecido un número de teléfono.
*   "Mostrar en móvil" Al activar esta opción, el icono de Whats soló se mostrara en dispositivos móviles.
*   "Llamada a l'acción" El mensaje que se introduzca se mostrara como un tooltip permanente al lado del icono.
*   "Mensaje" Si se introduce un mensaje, este sera predefinido para todos los que quieran usar el botón de Whats.
*   "Posición en pantalla" Se puede elegir la posición donde se mostrara el botón (Derecha o Izquierda).
*   "Establecer horario" Puedes establecer un horario de uso del botón, cuando este fuera del horario establecido, el botón se mostrara con una opacidad i no podrá ser usado.

== Installation ==

La instalacion de este plugin es igual a todos los de Wordpress.

e.g.

1. Subir el directorio `whats_submit` en la ruta `/wp-content/plugins/`
2. Activar el plugin en el apartado 'Plugins' del menú WordPress
3. Acceder al apartado WhatSubmit en el menú de Wordpress y empieza a configurar el plugin.

== Frequently Asked Questions ==

= ¿Recibo los mensajes en la propia web? =

No, el botón de Whats sólo sirve para iniciar la conversació, no te mostrara si has recibido un nuevo mensaje.

= ¿Responde un bot o un humano? =

Este plugin ésta diseñado para facilitar la interacción entre los usuarios y el propietario de la web.
Al estar usando la red social Whats, podemos afirmar, que responde un humano, manualmente, uno a uno.
